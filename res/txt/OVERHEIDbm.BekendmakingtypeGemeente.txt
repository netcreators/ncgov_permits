# Toelichting
# 1. Dit overzicht bevat een lijst van toegestane type bekendmakingen die gelden voor een gemeente. 
# 2. Dit overzicht is de basis voor de controle op de toegestane waarden van het schema 
#    "OVERHEID.bekendmakingtypeGemeente". Zie voor meer informatie en voorbeelden het 
#    IPM versie 3.0.
# 3. Bij het toevoegen van de content aan het metadataveld 'type bekendmaking' dient u één van 
#    deze type te selecteren.
# 4. Alle Ontheffingen en vrijstellingen zijn niet apart opgenomen in het overzicht. Dit is gedaan
#    omdat deze in principe te herleiden moeten zijn naar een vergunning of besluit. Omdat het
#    voor een burger wel relevante informatie kan zijn verzoeken wij de gemeenten om deze
#    termen in de metadata van de titel of in de omschrijving mee te geven.
# 5. Bij categorie vergunningen staat o.a. de 'apv vergunning' (verzameling van vergunningen). 
#    Daarnaast staan in deze categorie ook specifieke vergunnningen die onderdeel kunnen zijn 
#    van de apv vergunning. De metadata voor dit project moet zo specifiek mogelijk gekozen 
#    worden. Dit betekent dat wanneer de specifieke vergunning in de lijst voorkomt dat deze 
#    geselecteerd dient te worden. Wanneer de betreffende vergunning er niet tussen staat dan 
#    moet de 'keurvergunning' geselecteerd worden.
# 6. Alleen wanneer echt geen juist type bekendmaking geselecteerd kan worden mag voor
#    ‘overig’ gekozen worden.
# 7. Op verzoek kunnen wij deze woordenlijst ook in XML formaat (XSD) aanleveren. 
# 8. Wij verzoeken de gemeenten om fouten en suggesties in dit overzicht aan ons te melden. Komt u in de
#    praktijk een veel voorkomende type bekendmaking tegen die u nergens makkelijk kunt
#    onderbrengen, mail dit dan naar bekendmakingen@ictu.nl en geef ons daarbij een suggestie
#    hoe we het kunnen verbeteren.
#
#    Het projectteam bekendmakingen hoopt dat jullie met deze lijst uit de voeten kunnen en we
#    willen iedereen veel succes wensen bij de implementatie van het project binnen zijn of haar
#    organisatie.
